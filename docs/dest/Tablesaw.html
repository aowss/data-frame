Jupyter

Tablesaw can be used inside Jupyter notebooks using BeakerX.

  1. Run BeakerX using Docker : docker run -p 8888:8888 beakerx/beakerx

  2. Access Beaker at http://locahost:8888?token={generatedToken} [1] [2]

  3. Use the StartHere.ipynb notebook

  4. Scroll down to the Tablesaw section and use the notebook provided there

1. The URL is written to the console, as shown, when you run BeakerX
2. You might have to use localhost instead of the random id generated for the host, e.g. 4a25c59faf29 in the screenshot
